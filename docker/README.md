# Dockerfiles to run RS

| Folder                 | Details                                                                                                        |
| :--------------------- | :------------------------------------------------------------------------------------------------------------- |
| [_user](_user)         | Adds user to a docker image.                                                                                   |
| [realsense](realsense) | Generates a docker image to run RS. <br /> The command in the runfile is needed to run RS in docker container. |
